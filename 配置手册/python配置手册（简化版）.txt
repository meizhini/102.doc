# 参考 “Python踩坑之路-Python-3.6 安装pycrypto 2.6.1各种疑难杂症及解决方案” 安装pycrypto
1. 安装对应版本的msvs 例如Visual Studio 2015
2. 如果不需要用vs进行开发的话，装community免费版就可以了
3. 安装三个：Common Tools for Visual C++ 2015、python Tools for Visual StudioMarch 2016（）、Microsoft web Developer tools
记得勾选VC++2015和Python Tools。Web Deveolper Tools的话最好也勾上吧
4. 配置环境变量
变量名：VCINSTALLDIR
（变量值为vs安装路径下的VC，默认是这个）
变量值：C:\Program Files (x86)\Microsoft Visual Studio 14.0\VC
5. 安装pycrypto时需要设置临时的：set CL=/FI"%VCINSTALLDIR%\INCLUDE\stdint.h" %CL%


ibm需要下载驱动、pycrypto需要设置临时变量、qtpandas需要修改配置，读取版本已经处理的在py.rar文件中
